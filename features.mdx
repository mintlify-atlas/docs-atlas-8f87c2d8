CUCEIUbicate is a comprehensive mobile application designed specifically for CUCEI (Centro Universitario de Ciencias Exactas e Ingenierías) students at the Universidad de Guadalajara. The app provides interactive campus navigation, intelligent assistance, and essential academic services all in one place.

## Interactive Campus Navigation

The heart of CUCEIUbicate is its advanced map navigation system that helps students find their way around campus effortlessly.

### Map Features

- **Zoomable Campus Map**: Interactive SVG-based map with smooth zoom and pan controls
- **Building Search**: Quick search to locate any building or facility on campus
- **Point of Interest Markers**: Tap any building to view detailed information
- **Route Visualization**: Visual path display with coordinate-based routing

<Note>
The map supports both pinch-to-zoom and double-tap zoom gestures for optimal navigation experience.
</Note>

## AI-Powered Chatbot

The integrated chatbot provides intelligent assistance for all campus-related queries.

### Chatbot Capabilities

- **24/7 Availability**: Always ready to answer your questions
- **Campus Information**: Details about buildings, facilities, and services
- **Schedule Assistance**: Information about academic calendars and events
- **Smart Context**: Maintains conversation history for better responses
- **Link Detection**: Automatically formats URLs and email addresses

<Accordion title="View Chatbot Implementation">
The chatbot uses Google's Generative AI and Stack AI for intelligent responses:

```javascript Chatbot.js (lines 142-169)
async function query(user_input) {
  const payload = {
    user_id: "1234",
    "in-0": user_input,
  };

  try {
    const response = await fetch(API_URL, {
      method: "POST",
      headers: HEADERS,
      body: JSON.stringify(payload),
    });

    if (!response.ok) {
      throw new Error("Error al comunicarse con la API");
    }

    const responseData = await response.json();
    const rawResponse =
      responseData.outputs?.["out-0"] ||
      "No se recibió una respuesta válida en el campo 'out-0'.";

    return rawResponse.slice(0, 5000);
  } catch (e) {
    return `Error al comunicarse con la API: ${e.message}`;
  }
}
```
</Accordion>

### Conversation Features

- **Message Persistence**: Chat history saved locally with AsyncStorage
- **Auto-Scroll**: Smooth scrolling to latest messages
- **Copy to Clipboard**: Easily copy bot responses
- **Typing Indicators**: Visual feedback during response generation
- **Inactivity Management**: Automatic conversation cleanup after periods of inactivity

<Warning>
The chatbot will prompt to clear the conversation after 2 minutes of inactivity and automatically clear it after 3 minutes to maintain performance.
</Warning>

### Service Implementation

The School Services screen dynamically loads service information from JSON:

```javascript School_services.js (lines 48-86)
const School_services = () => {
  const [jsonData, setJsonData] = useState(null);
  const [refreshing, setRefreshing] = useState(false);

  useEffect(() => {
    loadJsonFromStorage();
  }, []);

  const loadJsonFromStorage = async () => {
    try {
      const storedData = await AsyncStorage.getItem("scholarServicesData");
      if (storedData !== null) {
        setJsonData(JSON.parse(storedData));
      } else {
        const json = require("../../../json/scholar_services.json");
        setJsonData(json);
        await AsyncStorage.setItem("scholarServicesData", JSON.stringify(json));
      }
    } catch (error) {
      console.error("Error loading data from storage:", error);
      const json = require("../../../json/scholar_services.json");
      setJsonData(json);
    }
  };

  const loadJson = async () => {
    setRefreshing(true);
    try {
      await new Promise((resolve) => setTimeout(resolve, 1000));
      const json = require("../../../json/scholar_services.json");
      await AsyncStorage.setItem("scholarServicesData", JSON.stringify(json));
      setJsonData(json);
    } catch (error) {
      console.error("Error refreshing data:", error);
    } finally {
      setRefreshing(false);
    }
  };
};
```
